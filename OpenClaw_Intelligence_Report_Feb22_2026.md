## What Is OpenClaw?

OpenClaw is NOT an AI model. It is an open-source runtime and message router that:

1. **Connects to any AI model** ‚Äî Claude, GPT-5.3, Gemini 3.1, Grok, MiniMax, Kimi, local via Ollama
2. **Runs on your own hardware** ‚Äî Mac Mini, Linux server, VPS, Android phone, Raspberry Pi
3. **Interfaces through existing messaging apps** ‚Äî WhatsApp, Telegram, Discord, Slack, iMessage, LINE, IRC, Signal, Teams, Feishu, Twitch
4. **Executes real actions** ‚Äî Browse web, manage email, run scripts, control smart devices, files
5. **Maintains persistent memory across sessions** ‚Äî With user-built Memory System 2.0
6. **Operates 24/7** ‚Äî Even when you're offline
7. **Supports HuggingFace models, custom providers, local models**

### The Key Insight

**ChatGPT is a website you visit. OpenClaw is an AI that lives on your server and works for you continuously ‚Äî even when you're asleep.**
